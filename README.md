# MUTAS - Frontend React

## Description
Frontend de l'application MUTAS - Gestion complète de sinistres avec interface moderne et responsive.

## Technologies

- **Framework** : React 18
- **Build Tool** : Vite
- **Styling** : Tailwind CSS
- **Routing** : React Router v6
- **Animations** : Framer Motion
- **Graphiques** : Chart.js / React-ChartJS-2
- **Icônes** : React Icons
- **Notifications** : React Hot Toast
- **PDF Generation** : jsPDF / jspdf-autotable
- **Excel Export** : XLSX

## Installation

### Prérequis
- Node.js 18+
- npm / yarn

### Étapes

1. Cloner le projet
git clone https://github.com/JuniorMartin237/mutas-frontend.git
2. Accéder au projet
cd mutas-frontend
3. Installer les dépendances
npm install
4. Créer le fichier .env
cp .env.example .env
5. Démarrer le serveur
npm run dev
